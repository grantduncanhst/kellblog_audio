# Podcast directory submission (one-time)

After `kellblog-audio publish` succeeds and `https://audio.kellblog.com/feed.xml` is live:

## Apple Podcasts

1. Go to [Apple Podcasts Connect](https://podcastsconnect.apple.com/).
2. Add a new show → **RSS Feed**.
3. URL: `https://audio.kellblog.com/feed.xml`
4. Verify ownership if prompted.
5. Set primary category: **Business** → **Management**.

## Spotify for Creators

1. Go to [Spotify for Creators](https://creators.spotify.com/).
2. **Add podcast** → paste the same RSS URL.
3. Spotify imports all episodes; new episodes appear when the feed updates (nightly Action).

## Optional

- Overcast: [podcast submission](https://overcast.fm/podcasterinfo)
- Pocket Casts: via [pocketcasts.com/submit](https://pocketcasts.com/submit)

## Validate feed first

- https://castfeedvalidator.com/
- https://podba.se/validate/

Upload local feed for testing: `output/feeds/feed.xml`
